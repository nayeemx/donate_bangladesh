DOM as a way for your browser to understand the HTML of a webpage. It lets developers add, remove, or change elements on a page dynamically, making web pages interactive. For example, in this project i change the donation and history button color using dom.
To select an element from the DOM, i can use JavaScript methods like:
document.getElementById('id'): Selects an element by its unique ID.
Example: document.getElementById('myButton')

document.querySelector('selector'): Selects the first element that matches a CSS selector.
Example: document.querySelector('.myClass') (selects the first element with class myClass)

document.querySelectorAll('selector'): Selects all elements that match a CSS selector.
Example: document.querySelectorAll('p') (selects all

elements)

Event delegation is a technique where instead of adding individual event listeners to multiple child elements, you add a single event listener to a parent element. The event bubbles up from the child to the parent, allowing the parent to handle it.

Why It's Useful:
1.Improves performance by reducing the number of event listeners.
